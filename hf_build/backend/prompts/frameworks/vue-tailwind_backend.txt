[FRAMEWORK-SPECIFIC INSTRUCTIONS: vue-tailwind backend]

- **CRITICAL STARTUP RULE**: You MUST write an Express server for the backend. Install `concurrently` and `cors`, and update `"dev"` in `package.json` to `"dev": "concurrently \"vite\" \"node src/server.js\""`. Ensure the Express server runs on port 3000 or 3001, and the Vue frontend connects to it.
